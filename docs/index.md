# QQ Chat Exporter（macOS 版）

QQ Chat Exporter（QCE）现在只保留 macOS 支持。

## 你会得到什么

- 菜单栏启动器
- Mac 风格 Web 界面
- 好友 / 群聊聊天记录导出
- HTML / JSON / TXT / Excel 导出

## 快速入口

- [使用手册](guide.md)
- [如何反馈问题](feedback.md)
- [如何贡献](contributing.md)

## 下载与启动

1. 解压 macOS 包
2. 双击 `打开真实QQ导出器.command`
3. 点击菜单栏图标打开网页

> 访问令牌保存在 `~/.qq-chat-exporter/security.json`。
